# Laravel Code
Setup
- take clone
- Vite is used so please run
```sh
npm install
```
- then create database with name  *Laravel*
- again run command
```sh
php artisan migrate
```

Finally run 
```sh
php artisna serve
```

dont forget to run 

```sh
npm run dev 
```

to start node services for Vite


Now it the route 
```sh
localhost:8000/register
```

to create account

Thats it
